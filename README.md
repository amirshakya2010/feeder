 - bower install feeder
 - OR Download directly

## Usage
  ## Used as directive
  -  "<feed></feed>"
  -  feed url = "https://myresearchesblog.wordpress.com/feed" feed-template-url = "path/to/template.html" no-result-text="Incase of no result found or failed case"></feed
